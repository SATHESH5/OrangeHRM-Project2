This is a project folder for "OrangeHRM website"

There are two folder:-

1.) Test_Codes # It consists of test case files (TC - Test Case files).

2.) Test_Data # It consists of test case data's i.e. username, password, XPATH, ID etc.

NOTE # If you want to run a Test Case File kindly go to the Test_Codes to run a specific Python Selenium Automation File.


COMMAND TO RUN A TEST FILE
--------------------------

pytest -v -s --capture=sys --html=/reports/reports.html <Test Case_File Name.py>
